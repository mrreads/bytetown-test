# Тестовое задание

Присуствует **адаптивная вёрстка**, минимальный ориентир **320px**.
## Использованные библиотеки

- GSAP + ScrollTrigger

## Запуск и сборка

``npm run dev`` - для запуска режима разработки

``npm run build`` - для сборки сайта, он будет располагаться в папке **dist**
